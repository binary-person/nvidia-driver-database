# frontend

Static SvelteKit frontend for browsing the derived `browser.sqlite` database in the browser with SQLite WASM and OPFS-backed worker storage.

Common commands:

```sh
npm install
npm run dev
npm run check
npm run test
npm run build
```

For the full repo-level setup, local database behavior, and GitHub Pages deployment flow, see [`../repo-setup.md`](../repo-setup.md).
